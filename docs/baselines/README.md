# 质量基线

基线用于发现工具输出或质量度量的非预期变化，不是临时测试文件。

| 目录 | 内容 |
|---|---|
| `manifests/` | 代表性 manifest v1 输入与 roundtrip 兼容样本 |
| `runtime/` | 隔离运行时场景、动作和断言 |
| `roundtrip/` | 组件库与原页面的结构/文本等价度结果 |

更新原则：

1. 仅在行为变化是有意且已解释时更新。
2. 同时更新相关测试和设计文档。
3. 不把带本机绝对路径、身份状态或远程依赖的结果写入基线。
4. runtime 场景中的输入值必须避免敏感信息，并保持错误输出脱敏。
