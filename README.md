# ui-dismantler

> 将自包含 HTML 案例页拆解为可复用组件库的工具集。

根据输入 HTML 的真实结构与样式，产出参数化的结构模板、归一化的样式系统、可重绑定的交互逻辑与数据契约，使任意结构的 HTML 都能被还原为数据驱动的可复用组件库。

## 现状

当前为 **v1 基线**（`src/skill/`），采用"识别已知模式 → 套预定义模板"的路线：
- 对明星组合类页面（tab + 成员网格 + 时间线 + 轮播）实现端到端自动化
- 8 项强约束校验（命名 / 变量 / 数据分离 / 响应式 / A11y / 主题可定制 / 零依赖 / 文档）
- 支持逐案例拆解与垂类聚合

v1 的局限：对非预定义结构的页面走 generic 兜底，产出"合规但空壳"的库。

## 路线

v2 转向"提模板留占位"：从 HTML 本身参数化抽象结构/样式/交互，不再依赖套预设模板。
详见 [docs/ROADMAP.md](docs/ROADMAP.md)。
