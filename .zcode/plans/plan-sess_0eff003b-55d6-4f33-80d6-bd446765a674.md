### Key design decisions
- Tab bar uses `role="tablist"` (NOT `<nav>`) â†’ avoids triggering page-level nav-panel on body, so panels get individually classified
- Splash overlay is outside the view-stack â†’ detected as a separate element
- Each panel triggers exactly one detector â†’ tests detector non-interference
- JS data uses multiple naming conventions (NODES, causeChain) â†’ exercises `ViewContext.scripts`
- Domain-neutral content (technical terms, not entertainment) â†’ no brand-name special-casing needed
