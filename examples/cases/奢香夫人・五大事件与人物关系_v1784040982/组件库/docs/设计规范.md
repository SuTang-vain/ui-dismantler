## 2. 结构元素

| 区块 | 类名 | 说明 |
| --- | --- | --- |
| 画布 | `.sg-frame` | flex 纵向，圆角 15px |

| 图例 | `.sg-graph-legend` | 左上角，4 类关系 SVG 线样 |

## 3. 交互模式

- **事件切换**：点击事件按钮 / 左右箭头 / 键盘方向键，重排图谱、更新简介与索引。
- **节点查看**：点击头像 → 高亮相关边并淡化其余 → 弹窗显示作为与影响。
- **节点拖动**：鼠标/触摸拖拽改变百分比坐标，连线实时重绘，首次拖动后隐藏提示。
- **弹窗关闭**：✕ 按钮 / 遮罩 / ESC。
- **连线高亮**：选中节点相关边 `hl`，其余 `dim`，无标签边自动隐藏。

## 4. 逻辑设置

- **响应式**：`@media(max-width:480px)/(max-height:480px)` 与 `@media(max-width:320px)/(max-height:365px)`，含 `min-width:769px` 事件按钮均布。
- **数据驱动**：事件、人物、连线、关系全部经 `options` 传入。
- **零依赖**：无外部 JS/CSS。
- **性能**：resize 防抖 120ms 重排连线；图片懒加载 + 失败回退渐变。

## 5. 无障碍

| 场景 | 实现 |
| --- | --- |
| 事件切换 | `role=tablist/tab`，`aria-selected`，键盘方向键 |
| 节点 | `role=button`，`tabindex=0`，`aria-label` 含姓名/角色/关系 |
| 弹窗 | `role=dialog`，`aria-modal`，ESC 关闭，焦点回退 |
| 状态公告 | `aria-live=polite` 公告事件切换与详情查看 |
| 装饰性元素 | 图谱提示、滑动提示 `aria-hidden=true` |

## 6. 字体与动效

- 系统无衬线：`-apple-system, BlinkMacSystemFont, "PingFang SC", "Microsoft YaHei", sans-serif`
- 动效缓动：`--sg-ease`（标准）/ `--sg-ease-back`（弹窗回弹）
- 动画：头像 `sg-avatar-breath` 2.4s 呼吸、节点 `sg-float-motion` 微浮动、提示 `sg-gh-pulse` / `sg-swipe-nudge` 脉冲

## 7. 组件清单

shexiang-furen.css 共定义约 55 个 `.sg-*` 类，覆盖：画布、事件简介、图谱（图例/画布/连线/节点/头像/名称/角色/提示）、事件切换条（按钮/箭头/滑动提示）、弹窗（头部/头像/信息/作为/影响/关闭）。
