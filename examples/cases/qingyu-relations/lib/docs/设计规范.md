### 1.2 语义分组

- **品牌主色组**：`--sg-primary` / `--sg-primary-dark` / `--sg-primary-light`，承担所有"选中 / 激活 / 主操作"语义。
- **文字灰阶组**：`--sg-ink` → `--sg-muted` → `--sg-subtle` 三级递减，对应主标题 → 副标 → 提示。
- **背景层次组**：`--sg-soft`（页面）→ `--sg-paper`（卡片）→ `--sg-stage`（兜底底色），形成页面 / 卡片 / 媒体底的三层。
- **关系语义组**：5 个 `--sg-rel-*` 变量，分别对应血缘 / 情感 / 师徒 / 盟友 / 对立，图谱连线、图例、详情标签共用一套色板。

### 1.3 渐变叠加

| 渐变 | 用途 | 定义 |
|---|---|---|
| `--sg-bg-gradient` | 页面背景 | `linear-gradient(160deg, #f5f8ff 0%, #eaeff9 60%, #dee6f4 100%)` |
| `--sg-card-gradient` | 主色卡片 / 标签 | `linear-gradient(135deg, var(--sg-primary), var(--sg-primary-light))` |

> 渐变中的颜色全部引用 `--sg-*` 变量，无硬编码 `#hex`。

## 2. Tab 元素

### 2.1 主 Tab（底部 3 视图）

| 属性 | 值 |
|---|---|
| 容器 | `.sg-tab-bar[role=tablist]` |
| 项 | `.sg-tab[role=tab][aria-selected][aria-controls][tabindex]` |
| 面板 | `.sg-view[role=tabpanel][aria-labelledby][aria-hidden]` |

三个主 Tab：`graph`（关系图谱）/ `mind`（剧情因果）/ `works`（作品推荐）。

| 属性 | 值 |
|---|---|

| 面板 | `.sg-works-grid[role=tabpanel][aria-labelledby]` |

四个分类：`series`（同系列）/ `theme`（同题材）/ `cast`（同主演）/ `ip`（同作者）。每页 4 个作品卡，超过则翻页。

## 3. 交互模式

### 3.1 视图栈

3 个 `.sg-view` 绝对定位堆叠于 `.sg-main`，通过 `.active` 切换 opacity + pointer-events。切换时：
- 进入 `graph`：测量画布 + 重排头像 + 启动浮动动画
- 离开 `graph`：暂停浮动动画
- 进入 `mind`：下一帧重渲染思维导图（尺寸可能已变）

### 3.2 关系图谱（视图 1）

**节点**：`.sg-node`（`<button>`，绝对定位，主角加 `.big`）
- **浮动动画**：`.sg-float-inner` 走 `@keyframes sg-float-motion`，每个节点随机 duration（6-10s）与 delay
- **拖拽**：mouse + touch，位移除以 scaleFactor 换算到画布逻辑坐标；拖动超 4px 标记 `dragged`，松开后屏蔽本次 click
- **点击**：打开右侧角色面板（PC）/ 居中弹窗（移动）
- **首次引导**：主角头像 `.tap-hint` 脉冲光环，5.6s 后移除

**连线**：`svg.sg-edges > line.sg-edge.<type>`
- 端点收缩到头像边缘（不遮挡头像），随浮动动画每帧更新
- 选中节点时：关联连线 `.hl`（高亮）、其余 `.dim`（淡化）

**角色面板**：`.sg-char-panel[role=dialog][aria-modal]`
- PC：右侧 300px 固定面板，画布让位压缩
- 移动（≤768 容器）：居中弹窗 + 遮罩，图例隐藏
- 内容：头像 + 姓名 + 演员 + 与主角关系标签 + 人物简介
- 关闭：× 按钮 / ESC（全局）

### 3.3 剧情因果（视图 2）

**横向思维导图**：根节点（`.sg-mind-node.root`）+ N 个剧情卡片横向排列
- 卡片：缩略图 + 类型徽章 + 集数 + 标题 + 因果（causeNext）
- 连线：贝塞尔曲线 `path.sg-mind-link`
- 折叠：每个非末位节点右侧 `+`/`−` 按钮，点击切换 `mindCutoff` 重渲染

**底部滑杆**：`.sg-msl-track[role=slider]`
- 拖动 thumb / 点击轨道 / 点击点位 → 滚动到对应节点
- 滚轮：横向滚动（deltaY/deltaX 换算）
- 移动端触摸：左右滑动（判定横向手势才接管，纵向交还页面）
- 键盘：←/→ 滚动 10%
- 滑动结束后 80ms 内屏蔽卡片点击，避免误开弹窗

**剧情弹窗**：`.sg-plot-modal[role=dialog][aria-modal]`
- 封面 + 集数徽章 + 标题 + 剧情解析 + 关键人物头像列表
- 打开时聚焦关闭按钮；关闭：× / 遮罩 / ESC

### 3.4 作品推荐（视图 3）

**网格**：`.sg-works-grid`，PC 4 列单行、移动 2 列多行滚动
- 卡片：封面（含集数 tag）+ 名称 + 主演（取前 2 位）+ 推荐理由
- `upcoming` 类：待播作品 tag 变金色
- 翻页：上一页 / 下一页按钮，禁用态 `disabled`，单页时隐藏 pager

## 4. 逻辑设置

### 4.1 响应式断点

| 档位 | 触发 | 画布基准 | 关键调整 |
|---|---|---|---|
| PC | 默认 | 788×492 | 完整布局、3D 浮动、右侧面板 300px |
| WISE | `max-width:500px` | 380×456 | 角色面板转居中弹窗、卡片窄、tab 紧凑 |
| 极端 | `max-width:320px` | 380×456 | 字号极致压缩、图例 9px、卡片 132px |

> 画布采用**等比缩放**：JS 按窗口尺寸选基准（788×492 或 380×456），`transform: scale()` 等比填充视口，内容坐标始终在基准坐标系内，保证布局一致。

### 4.2 等比缩放策略

```
window 尺寸 → 选 base（≤500 取 mobile 380×456，否则 desktop 788×492）
scale = min(w/base.w, h/base.h)
.sg-frame { width: base.w; height: base.h; transform: translate(offset) scale(scale); }
```

所有 `getBoundingClientRect()` 取到的视觉像素都需 ÷scaleFactor 换算回画布逻辑坐标（拖拽 / 连线 / 滑杆均遵循）。

### 4.3 数据驱动

- 所有可变内容（角色 / 连线 / 剧情 / 作品）经 `options` 传入，**禁止 JS 硬编码业务文案/URL**
- 图片路径 = `options.imgBase + chars[key].img`，`imgBase` 可配置
- 缺失 `layoutRel` 时自动环形布局（主角居中，其余均匀环绕）

### 4.4 A11y

| 元素 | 实现 |
|---|---|
| 主 Tab | `role=tablist/tab/tabpanel` + `aria-selected/controls` + roving tabindex + ←/→/Home/End |
| 子 Tab | 同上，独立 tablist |
| 角色面板 | `role=dialog` + `aria-modal=true` + `aria-labelledby` |
| 剧情弹窗 | `role=dialog` + `aria-modal=true` + `aria-labelledby` + 打开聚焦关闭按钮 |
| 滑杆 | `role=slider` + `aria-valuemin/max/now` + `aria-label` + 键盘 ←/→ |
| 图标按钮 | `aria-label`（关闭 / 上一页 / 下一页 / 折叠展开） |
| 关闭 | ESC 全局关闭（剧情弹窗优先于角色面板）+ 遮罩点击 + × |
| 播报区 | `.sg-sr-only[aria-live=polite]`，切换 tab / 分类 / 打开详情时播报 |
| 图例 | `aria-hidden=true`（装饰性，色板已由标签文字承载语义） |

### 4.5 性能稳健性

- 浮动动画：纯 CSS `@keyframes`，仅 `transform` 变化，GPU 加速
- 连线跟随：`requestAnimationFrame` 循环，仅在 `graph` 视图激活时运行
- 拖拽：`will-change: left, top` + 拖动时 `transition: none`（`.dragging-now`）
- resize：缩放层立即同步（零成本），重排 debounce 150ms，仅基准切换时才重建图谱/思维导图
- 图片预热：可选（原案例在 load 时预加载所有图片，本库按需加载 + 兜底底色）
