# 设计规范 — 鳡·特征互动观察

## 1. 主题色系统

### 设计令牌

| Token | 值 | 语义用途 |
|---|---|---|
| `--sg-primary` | `#6487FA` | 主功能色：热点/激活态/主按钮/主角高亮/雷达主条 |
| `--sg-primary-dark` | `#3D5FD1` | 主色按下/主角激活渐变深端 |
| `--sg-primary-light` | `#7A9BFF` | 主色渐变浅端 |
| `--sg-primary-mist` | `#EEF3FF` | 主色雾化浅底（主角按钮渐变起） |
| `--sg-primary-haze` | `#DDE6FF` | 主色雾化深底 |
| `--sg-primary-mid` | `#5A7AF0` | 主色中调 |
| `--sg-ink` | `#1E1F24` | 主文字 |
| `--sg-muted` | `#848691` | 次级文字 |
| `--sg-subtle` | `#B7B9C1` | 三级文字 |
| `--sg-paper` | `#FFFFFF` | 卡片/面板底 |
| `--sg-stage` | `#F8F8F8` | 画布/参数表行底 |
| `--sg-line` | `rgba(183,185,193,.7)` | 分割线/边框 |
| `--sg-line-strong` | `rgba(183,185,193,.95)` | 坐标轴/刻度尺 |
| `--sg-soft` | `rgba(100,135,250,.10)` | 主色浅底 |

### 语义分组

- **品牌**：`--sg-primary` / `--sg-primary-dark` / `--sg-primary-light` — 渐变用于激活态按钮、雷达主条
- **文字三级**：ink > muted > subtle
- **背景两级**：paper（面板）> stage（画布/参数行）
- **分割两级**：line（普通边框）> line-strong（坐标轴/刻度尺）

### 渐变叠加

- 激活态按钮：`linear-gradient(180deg, primary-light → primary)`
- 主角激活按钮：`linear-gradient(180deg, primary-mid → primary-dark)`
- 雷达主条：`linear-gradient(90deg, primary-light → primary)`
- 雷达目标条：`linear-gradient(90deg, subtle → muted)`
- 普通按钮底：`linear-gradient(180deg, paper → soft)`
- Modal 布局卡：`linear-gradient(140deg, soft → paper → stage)`
- 图片兜底：`linear-gradient(135deg, primary → primary-dark)`

## 2. 结构元素

### 画框 `.sg-frame`
PC 788×492，padding 8px，gap 8px；满屏时 padding 6px、gap 6px。

### 主区 `.sg-main` + 双视图 `.sg-view`
绝对定位叠加切换（opacity + scale）。`role=tabpanel`。

### 特征视图
- 刻度尺 `.sg-ruler`：左侧 56px，5 档刻度
- 长度轴 `.sg-length-axis`：底部水平线
- 视角切换 `.sg-pose-tabs`：`role=tablist`，pill 按钮（正面/侧面）
- 主图舞台 `.sg-lynx-stage`：`role=tabpanel`，按视角显隐 figure
- 热点 `.sg-hotspot`：圆形脉冲动画（`translate(-50%,-50%)` 居中于坐标点），`role=button`，按视角动态渲染

### 对比视图
- 视觉面板 `.sg-panel-visual`：
  - 详细对比按钮 `.sg-compare-detail-btn`（仅移动端可见）
  - 对比舞台 `.sg-compare-stage`：基线行 + 目标行 + 共享坐标轴 + 免责声明
  - 对比行 `.sg-compare-row`：grid 48px + 1fr
  - 图轨 `.sg-compare-track`：高 140px（PC）/100px（移动）
  - 对比动物 `.sg-compare-animal`：固定宽度 60.6%，img `transform-origin:left center` + scaleX 按体长比例缩放
- 详情面板 `.sg-panel-detail`：
  - 抽屉把手 `.sg-drawer-handle`（仅移动端）
  - 数据区 `.sg-data-section`：雷达图 + 参数表
  - 雷达图 `.sg-radar`：纯 CSS 双层条形图（主角 + 目标，5 维）
  - 参数表 `.sg-stats-list`：grid 4 列

### 物种切换栏 `.sg-switch-bar`
高 76px（PC）。`.sg-switch-group` 卡片内 `.sg-switch-items` 横排按钮。
- 主角按钮 `.sg-switch-item.feature`：flex 1.1，双层边框 + 深渐变
- 对比物种按钮 `.sg-switch-item`：flex 1，头像 + 名称，初始默认对比物种激活

### Modal `.sg-modal-overlay`
`role=dialog` + `aria-modal=true`。双模式：详情（放大图 + 文本）/ 对比（参数行列表），由 `.is-compare` 类切换。放大图按 zoomLeft/zoomTop 计算 transform。

## 3. 交互模式

| 模式 | 细节 |
|---|---|
| 视图切换 | 点击底部栏；键盘 ← → 跨视图；`aria-selected` 联动 |
| 视角切换 | 点击 pose tab；`role=tab` + `aria-selected` + `aria-controls`；切换后重渲染热点 |
| 热点详情 | 点击/Enter 热点 → Modal 放大图 + 说明；放大图按 zoomLeft/zoomTop 计算 transform |
| 对比物种切换 | 点击对比物种按钮 → 重渲染目标图/雷达图/参数表/scaleX 布局；aria-live 播报 |
| 详细对比 | 点击「详细对比」→ Modal 参数行列表 |
| 抽屉（移动端） | 点击把手展开/收起对比详情面板 |
| 关闭 Modal | × / 遮罩 / ESC；打开聚焦关闭按钮 |

## 4. 逻辑设置

### 响应式断点

| 断点 | 画布 | 布局变化 |
|---|---|---|
| PC 默认（≥600px 且 ≥420px 高） | 788×492 | 双面板横排，图轨 140px，对比图 60.6% 宽 + scaleX |
| `≤600px` 或 `≤420px` 高 | 满屏 | 单列，图轨 100px，对比图宽度按 110/140 比例缩放，详情改抽屉 |
| `≤340px` 或 `≤380px` 高 | 满屏 | 字号/按钮/卡片进一步收紧，Modal 紧凑布局 |

### 数据驱动
所有业务文案/图片/物种/热点/维度/分数/图元数据走 `options`，引擎零硬编码。

### A11y
- 视图 `role=tabpanel`，物种切换栏 `role=tab` + `aria-selected` + `aria-controls`
- 视角切换 `role=tablist`/`tab`
- 热点 `role=button` + `tabindex` + `aria-label`
- Modal `role=dialog` + `aria-modal=true`，ESC 关闭，打开聚焦关闭按钮
- 物种切换 `aria-live=polite` 播报

### 性能稳健性
- 对比图布局 resize 防抖 150ms
- 雷达条形图 `requestAnimationFrame` 触发宽度过渡
- 图片 `loading=lazy` + `decoding=async` + load/error 兜底

## 5. 字体排版

- 字体栈：`-apple-system, BlinkMacSystemFont, "Segoe UI", "PingFang SC", "Microsoft YaHei", sans-serif`
- 标题层级：modal-title 22px/800 → switch-name 15px/800 → pose-tab 14px/800
- 正文：14-15px；辅助 9-13px
- 圆角体系：frame 14px / panel 14px / button 12px / pill 999px / modal 20px
