| 区块 | 类名 | 说明 |

## 3. 交互模式

- **视图切换**：底部 Tab 切换图谱 / 作品，图谱切入时重排。
- **剧情切换**：点击剧情按钮 → 更新描述 → 重排图谱节点与连线。
- **节点布局**：主角居中，其余角色环形分布；坐标按画布百分比。
- **节点查看**：点击头像 → 打开右侧面板 → 高亮相关边、淡化其余。
- **节点拖动**：鼠标/触摸拖拽改变百分比坐标，连线实时重绘。
- **作品翻页**：每页 4 部，左右翻页按钮（移动端横滑）。
- **面板关闭**：✕ / ESC（移动端遮罩点击）。

## 4. 逻辑设置

- **响应式**：`@media(max-width:500px)` 与 `@media(max-width:320px)`，含面板全屏遮罩、作品横滑。
- **数据驱动**：人物、关系、剧情、作品全部经 `options` 传入。
- **零依赖**：无外部 JS/CSS。
- **性能**：resize 防抖 150ms 重排图谱；图片懒加载 + 失败回退渐变占位。

## 5. 无障碍

| 场景 | 实现 |
| --- | --- |
| 视图导航 | `role=tablist/tab`，`aria-selected`，视图 `role=tabpanel` |
| 剧情切换 | `role=tablist/tab` |
| 作品分类 | `role=tablist/tab` |
| 节点 | `role=button`，`tabindex=0`，`aria-label` 含姓名/演员 |
| 角色面板 | `role=dialog`，ESC 关闭，焦点回退 |
| 状态公告 | `aria-live=polite` 公告视图/剧情切换与详情查看 |

## 6. 字体与动效

- 系统无衬线：`-apple-system, BlinkMacSystemFont, "PingFang SC", "Microsoft YaHei", sans-serif`
- 动效缓动：`--sg-ease cubic-bezier(0.4,0,0.2,1)`
- 时长：`--sg-dur-fast 0.2s` / `--sg-dur-mid 0.28s` / `--sg-dur-slow 0.4s`

## 7. 组件清单

dahua-xiyou.css 共定义约 60 个 `.sg-*` 类，覆盖：画布、主区视图栈、底部 Tab、剧情切换条、图谱（图例/画布/连线/节点/头像/名称）、角色面板（头部/头像/信息/关系标签/描述/关闭）、作品区（分类/翻页/网格/卡片）。
