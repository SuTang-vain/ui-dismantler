# 设计规范

## 主题色

主色为 `--sg-primary: #6487fa`，搭配浅蓝光晕和粉色径向背景；正文、次级文字、边界和卡片背景均通过 `--sg-*` 变量控制。

## 结构

- `sg-costar-view`：演员关系图，包含 SVG 边、中心节点和演员节点。
- `sg-identity-view`：艺人身份信息、画廊、标签和事实云。
- `sg-modal`：作品与演员详情弹层。
- `sg-nav`：两个视图的底部导航。

## 交互

支持导航切换、画廊翻页/滑动、关系节点拖动、演员详情弹层、ESC 关闭和焦点约束。

## 响应式

组件运行时根据宿主容器选择 788×492 或 380×456 基准，再按容器尺寸缩放；极小画布使用 `sg-is-tiny-canvas` 状态。

## 可访问性

导航具有 `aria-label` 和 `aria-current`，弹层由运行时维护 `aria-hidden`、焦点回收和键盘 ESC 关闭。
