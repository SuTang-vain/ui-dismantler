# 明星组合 · 动态百科成员探索卡 - 设计规范

> 源自 `BLACKPINK v10 最新迭代版` 案例页（PC 788×492 / WISE 380×456 / 极端小屏 280×340），用于"明星组合"垂类的统一组件库建设。
>
> 本规范对应组件库 v1.2.0，在 v1.1（gold-standard）基础上新增「初始引导遮罩」.sg-entry-cover 特性。

## 一、主题色系统

### 1.1 设计令牌（CSS Custom Properties）

所有颜色均通过 `:root` / `.sg-frame` 变量定义，便于主题切换与复用。原案例变量归一化到 `--sg-*` 命名空间，共 11 个主题色令牌：

| 原变量 | 归一化 | 值 | 语义 / 用途 |
|---|---|---|---|
| `--ink` | `--sg-ink` | `#1E1F24` | 主文字 / 头像兜底深色背景 |
| `--muted` | `--sg-muted` | `#848691` | 次级文字（副标题、说明、故事正文） |
| `--subtle` | `--sg-subtle` | `#B7B9C1` | 三级文字（占位、计数 `<small>`、引导提示） |
| `--line` | `--sg-line` | `#ECEDF1` | 分割线 / 卡片描边 / 未激活指示点 |
| `--paper` | `--sg-paper` | `#FFFFFF` | 卡片底色 / 主面板背景 / 故事面板底 |
| `--stage` | `--sg-stage` | `#F8F8F8` | 画布舞台背景（body） |
| `--primary` | `--sg-primary` | `#6487FA` | **品牌主色** - 选中描边、激活态、主按钮、箭头、时间轴节点、kicker |
| `--accent` | `--sg-accent` | `#E94F76` | **强调色** - "其它"Tab、创作故事引导点、年份高亮、故事关闭 hover |
| `--accent-2` | `--sg-accent-2` | `#111111` | 备用深色强调（预留） |
| `--soft` | `--sg-soft` | `#F2F4FB` | 主色浅底 - 选中卡片背景、hover 背景、展开态时间线节点 |
| `--soft-pink` | `--sg-soft-accent` | `#FFF0F4` | 强调色浅底（预留） |

另外在 `:root` 定义了以下派生令牌（非业务色，用于纯白/极浅中性底与渐变端点）：

| 令牌 | 值 | 用途 |
|---|---|---|
| `--sg-note` | `#90929a` | 声明/来源说明文字（介于 muted 与 subtle） |
| `--sg-white` | `#FFFFFF` | 纯白（hover 反白、文字浮于深色背景） |
| `--sg-neutral-soft` | `#FAFAFB` | 成员卡 / 时间线卡 / Modal 行底 |
| `--sg-ink-soft` | `#2A2B33` | 头像兜底渐变起点（比 ink 略浅） |
| `--sg-panel-top` | `#FBFBFD` | 详情面板渐变顶 |
| `--sg-panel-bottom` | `#F5F7FC` | 详情面板渐变底 |

### 1.2 语义分组

- **文字层级**：`--ink` > `--muted` > `--subtle`（三级灰阶）+ `--note`（声明文字）
- **品牌双色调**：`--primary`（蓝紫，主功能色）+ `--accent`（粉红，"其它/故事"语义色）
- **选中态**：边框 `--primary` + 背景 `--soft`（蓝紫浅底组合）
- **强调态**：圆点 `--accent` + 文字 `--accent`

### 1.3 渐变与叠加

| 用途 | 值 |
|---|---|
| 详情面板底色 | `linear-gradient(180deg, var(--sg-panel-top) 0%, var(--sg-panel-bottom) 100%)` |
| 成员卡底部信息蒙版 | `linear-gradient(to top, rgba(0,0,0,0.75), transparent)` |
| 作品卡底部信息浮层 | `linear-gradient(transparent, rgba(0, 0, 0, 0.82))` |
| 头像兜底渐变 | `linear-gradient(135deg, var(--sg-ink-soft) 0%, var(--sg-ink) 100%)` |
| Modal 遮罩 | `rgba(30, 31, 36, 0.45)` |
| 引导遮罩玻璃块底 | `linear-gradient(135deg, rgba(255,255,255,0.18) 0%, rgba(30,31,36,0.10) 50%, rgba(255,255,255,0.06) 100%)` |

> 渐变中的 `rgba(0,0,0,0.x)` 为纯黑白蒙版，按 spec 允许保留；其余颜色一律走 `var(--sg-*)`。

## 二、Tab 元素

### 2.1 结构

顶部 4 等分 Tab Bar，使用 `grid-template-columns: repeat(var(--sg-tab-count, 4), 1fr)`：

```
[ 成员详情 <small>4</small> ] [ 经历 <small>6</small> ] [ 团体作品 <small>6</small> ] [ 其它 ↗ ]
```

- **前 3 个 Tab**：切换面板，激活时 `border-bottom: 2px solid var(--sg-primary)`，文字由 `--muted` -> `--ink`
- **第 4 个 "其它" Tab**（`.sg-tab-more`）：不切换面板，打开 Modal；激活态使用 `--sg-accent` 下划线，未激活时尾部显示 `↗` 图标（CSS `::after`）

### 2.2 Tab 属性约定

- `role="tab"` / `aria-selected` / `aria-controls`
- "其它" Tab 额外维护 `aria-expanded`（Modal 开关同步）
- 计数 `<small>` 显示成员/经历/作品数量，激活时颜色由 `--subtle` 升为 `--muted`
- 切换面板时：同步 `aria-selected`、`hidden`、`.active` 类
- **初始引导态**：未点 Tab 前，所有 Tab `aria-selected=false`（遮罩可见，无内容页激活）

### 2.3 尺寸

| 场景 | 高度 | 字号 | 内边距 |
|---|---|---|---|
| PC | 36px | 12.5px / 600 | 0 |
| WISE ≤500px | 32px | 11.5px | 0 |
| 极端 ≤320px | 36px | 10px | 0 4px（`<small>` 隐藏） |

## 三、交互模式

### 3.1 视图栈（View Stack）+ 初始引导遮罩（v1.2 新增）

`.sg-view-stack` 内多个 `.sg-view` 绝对定位堆叠（`position:absolute; inset:0`），仅 `.active` 显示（`display: grid`）。Tab 切换通过显隐完成，无路由。

**初始引导遮罩** `.sg-entry-cover`（忠于实际 HTML v10-latest）：

- 页面加载时遮罩覆盖整个 view-stack，z-index 20，组合照铺满 + 玻璃磨砂四分入口
- 四列与 Tab 一一对齐（`.sg-entry-grids` 用 `grid-template-columns: repeat(var(--sg-tab-count), 1fr)`，`padding-top: 36px` 预留 tab-bar 高度）
- 每个玻璃块 `.sg-entry-cell` 含：编号（01-04）、名称、箭头（前三个 ↓，末位 ↗）
- **交互行为**：
  - 点击前三个玻璃块：触发对应 Tab 的点击逻辑 + 淡出遮罩
  - 点击「其它」玻璃块：恢复初始态（清空三个内容 Tab 选中态 + 显示遮罩）+ 打开资料 Modal
  - 点击 tab-bar 的内容 Tab（成员/经历/作品）：切换面板 + 淡出遮罩
  - 点击 tab-bar 的「其它」Tab：恢复初始态 + 显示遮罩 + 打开 Modal
- 淡出动效：`opacity 1->0`（.is-hidden）+ 420ms 后 `display:none`；恢复时先 `display:''` 再触发重排移除 .is-hidden，确保过渡生效

### 3.2 成员视图（Members）

- **2×2 网格**展示成员头像卡，每页 4 人
- **点击成员卡**：切换右侧"详情面板"内容（成员↔团体关系）；小屏（≤500 或 ≤380h）弹出成员 Modal（含头像信息头 + 关系列表）
- **自动播放**：3 秒/个循环切换选中成员，任意用户交互（click/touchstart）后停止
- **轮播控件**：左右圆形箭头（`--primary` 填充、hover 反白）、底部圆点指示器（激活态拉长为胶囊）
- **头像容错**：`load` 加 `.is-loaded`、`error` 加 `.is-error`；兜底显示 `avatar-fallback`（首字母 + 深色渐变）

### 3.3 经历视图（Timeline）- 原地展开故事

v10 的核心特性：**所有屏幕尺寸下，点击时间线条目原地展开经历背景故事**，不弹 Modal。

- **横向滚动轨道**（`scroll-snap-type: x mandatory`），PC 每页 3 条 / 小屏 2 条
- **PC 端**：箭头绝对定位在轨道左右两侧（`.sg-tl-prev-pc` / `.sg-tl-next-pc`）
- **移动端**：底部控制栏（左箭头 + 圆点指示器 + 右箭头，`.sg-tl-controls`）
- **页码标签**：`第 X / Y 页 · 关键节点`，随滚动实时更新
- **条目结构**：
  - 折叠态：图片 + time + title + 简短 desc（CSS 隐藏）+ "了解背景 ›" 引导提示
  - 展开态（`.is-expanded`）：显示 `.sg-t-story`（经历背景长文）+ 故事头（time+title重复）+ "经历背景" 标签 + 长文 + "收起" 按钮
- **展开动效**：
  1. 添加 `.is-expanded`，卡片宽度过渡到 58%（PC）/ 82%（WISE）/ 88%（极端）
  2. 立即关闭 `scroll-snap-type`（避免吸附干扰定位）
  3. 420ms 后（过渡完成）计算居中偏移并 `scrollLeft += offset`
  4. 恢复 `scroll-snap-type: x proximity`
- **关闭方式**：再点卡片 / 点"收起"按钮 / ESC 键
- **布局差异**：PC 展开态为横向（图片 30% + 故事 70%）；WISE/极端为纵向堆叠

### 3.4 作品视图（Works）- 中心聚焦 3D 轮播

- **5 档位布局**（`perspective: 900px`）：
  - `is-center`：280×280，`scale(1) translateZ(50px)`，z-index 3
  - `is-prev-side` / `is-next-side`：132×78%，`scale(.82) translateX(±180px) translateZ(-20px)`，opacity .5，blur 1.5px
  - `is-prev-far` / `is-next-far`：100×62%，`scale(.68) translateX(±312px) translateZ(-45px)`，opacity .28，blur 2.5px
- **自动播放**：3.5 秒/次，进入 Tab 启动、离开停止
- **交互**：
  - 点击侧卡 -> 切换为中心
  - 点击中心卡 / 点击"展开创作故事"按钮 -> 滑入故事面板
  - PC 鼠标悬停轮播暂停，移出恢复
  - ESC 关闭故事面板
- **故事面板**：覆盖作品视图，封面（flex 6）+ 文字（flex 4），移动端纵向堆叠

### 3.5 Modal 体系

| Modal | 触发 | 内容 |
|---|---|---|
| `#sg-modal`（资料说明） | "其它"Tab 或"其它"玻璃块 | 2 列事实网格（词条类型/经纪/出道/成员…） |
| `#sg-member-modal`（成员详情） | 小屏点成员卡 | 头像信息头 + 成员关系列表 |

- 统一圆形 X 关闭按钮（`.sg-modal-x-btn`，`rgba(0,0,0,0.45)` 底）
- 关闭方式：X 按钮 / 点击遮罩 / ESC 键
- 资料说明 Modal 底部含声明文字，无额外关闭按钮（关闭走 X/遮罩/ESC）

## 四、逻辑设置

### 4.1 响应式断点

| 断点 | 画布 | 关键变化 |
|---|---|---|
| 默认（PC） | 788×492 | 双栏（成员+详情）、时间线 3 列/页 + 展开态 58% 宽、作品 5 档位 |
| `≤500px`（WISE） | 380×456 | 单栏、详情面板隐藏、时间线 2 列 + 底部控制栏 + 展开 82%、作品百分比尺寸 |
| `≤320px 或 ≤380h`（极端） | 100%×100%（min 280×340） | 隐藏 `<small>`、单栏成员、点击成员弹 Modal、时间线 2 列 + 展开 88% |

### 4.2 自动播放策略

| 模块 | 间隔 | 启停条件 |
|---|---|---|
| 成员轮播 | 3000ms | 进入即启动；任意用户交互（click/touchstart）停止 |
| 作品轮播 | 3500ms | 进入作品 Tab 启动；离开/打开故事面板/悬停(PC)停止 |

### 4.3 数据驱动

- **成员数据**：`options.members` 数组，含 `key/name/role/shortName/state/img/photoSource/relations[]`
- **经历数据**：`options.timeline` 数组，含 `time/title/desc/img/alt/story`
- **作品数据**：`options.works` 数组，含 `img/alt/year/title/desc/story`
- **资料数据**：`options.moreFacts` 数组，含 `label/value/full?`
- **引导遮罩数据**：`options.entryCoverImg/entryCoverAlt/entryCells[]`（v1.2 新增）
- 所有可变内容走 `options`，渲染引擎（`star-group.js`）零硬编码业务文案/URL

### 4.4 无障碍（A11y）

- Tab/Tabpanel 完整 `role` + `aria-selected` + `aria-controls` + `aria-labelledby`
- "其它" Tab 额外维护 `aria-expanded`（Modal 开关）
- 成员卡 `aria-pressed` 表示选中态
- 详情面板 `aria-live="polite"` 动态播报
- Modal `role="dialog"` + `aria-modal="true"` + `hidden`
- 引导遮罩 `aria-hidden` 跟随显隐同步
- 所有图标按钮均有 `aria-label`
- ESC 键关闭优先级：时间线展开 > 成员 Modal > 故事面板 > 资料 Modal

### 4.5 性能与稳健性

- 图片 `loading="lazy"`，加载/错误态驱动兜底显示
- 滚动容器 `scrollbar-width: none` + `::-webkit-scrollbar { display: none }`
- `overscroll-behavior: none` + `touch-action: manipulation` 防误触
- Tab 切换时 `resetTimelineScroll()` 重置滚动并用 `requestAnimationFrame` 二次校准
- 几何计算 `_tlItemWidth()` 包含 gap（+10），优先取未展开卡片避免展开态干扰分页
- `requestAnimationFrame` 有兜底（`setTimeout`），jsdom 等无 rAF 环境也能运行
- 引导遮罩淡出用 setTimeout(420ms) 配合 transition，恢复时触发重排确保过渡生效
