## 3. 交互模式

- **视图切换**：顶部导航点击切换，图谱视图在切入时自动 `layout` 重排。
- **出处注解**：原文中 `[key]词[/key]` 渲染为可点击高亮，弹出 modal 显示注解。
- **图谱交互**：节点点击 → 侧栏详情；移动端 → 弹窗；筛选切换重排连线（dashed 流动动画）。
- **测验流程**：作答锁定 → 显示对错与反馈 → 解锁「下一题」→ 末题「查看结果」→ 结果页。
- **弹窗关闭**：点击「知道了」/ 遮罩 / ESC 键。

## 4. 逻辑设置

- **响应式**：`@media(max-width:500px)` 全屏堆叠 + 子标签；`@media(max-width:320px)` 收紧。
- **数据驱动**：所有可变内容（词条、释义、典故、节点、题目）经 `options` 传入，DOM 不含业务文案。
- **零依赖**：不引用任何外部 JS/CSS（字体 CDN 除外）。
- **性能**：窗口 resize 防抖 120ms 重排图谱；图片懒加载 + 失败回退渐变占位。

## 5. 无障碍

| 场景 | 实现 |
| --- | --- |
| 视图导航 | `role=tablist/tab`，`aria-selected`，面板 `role=tabpanel` |
| 子标签 | `role=tablist/tab` |
| 图谱节点 | `role=button`，`tabindex=0`，`aria-label` 含关系与词条 |
| 测验选项 | `role=button`，`tabindex=0`，键盘 Enter/Space |
| 弹窗 | `role=dialog`，`aria-modal=true`，ESC 关闭，焦点回退 |
| 状态公告 | `aria-live=polite` 公告切换/答题状态 |
| 反馈 | `.sg-qz-fb` 为 `role=status` |

## 6. 字体

- 系统无衬线：`-apple-system, BlinkMacSystemFont, "PingFang SC", "Microsoft YaHei", sans-serif`
- 出处原文衬线：`"Songti SC", "SimSun", serif`
- 层级：词条 40px / 题目 15.5px / 正文 12.5–14px / 关系标注 10–11px

## 7. 组件清单

xie-tianzi.css 共定义约 60 个 `.sg-*` 类，覆盖：画布、导航、面板、词条卡、核心信息网格、例句、典故卡、出处原文、图谱画布/节点/连线/筛选/侧栏、测验顶栏/题目/选项/反馈/结果、弹窗、Toast、子标签。
