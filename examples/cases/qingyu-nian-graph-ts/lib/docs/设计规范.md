# 设计规范

## 主题色

组件以墨色文字、暖白纸张背景、金棕强调色和关系类型色构成人物图谱视觉系统。所有主题色经过 `--sg-*` 变量归一化。

## 结构

- `sg-graph-view`：人物节点、SVG 连线和关系图例；
- `sg-char-panel`：人物简介及与主角的关系；
- `sg-works-view`：作品推荐分类与卡片分页；
- `sg-tabs`：关系图谱和作品推荐主视图切换。

## 交互

人物节点点击后显示人物详情；主 Tab 切换关系图和作品推荐；推荐区支持分类切换与前后翻页。

## 响应式

桌面端保持图谱与详情并列，900px 以下调整布局，768px 和 400px 以下进一步压缩节点、卡片和控制区。

## 可访问性

交互按钮保留可读文本或 aria-label；人物详情提供关闭控制；图片提供 alt；键盘可访问主要按钮。
