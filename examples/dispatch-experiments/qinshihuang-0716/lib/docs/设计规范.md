# 秦始皇关系图谱设计规范

## 主题色

以 `--sg-primary`、`--sg-ink`、`--sg-muted`、`--sg-line`、`--sg-paper` 为核心令牌，关系类型使用 `--sg-rel-family`、`--sg-rel-ruler`、`--sg-rel-ally`、`--sg-rel-enemy`。

## 组件清单

- `sg-event-intro`：事件简介和摘要。
- `sg-graph-wrap` / `sg-graph-canvas`：人物关系图谱与 SVG 连线。
- `sg-event-bar` / `sg-event-btn`：七大事件切换。
- `sg-modal-overlay` / `sg-modal-content`：人物详情弹窗，支持 Escape 关闭。

## 响应式

桌面采用大画布，小于 768px 切换紧凑排布，≤500px 与 ≤320px/低高度提供 WISE 与极端视口降级。

## 交互与无障碍

头像点击打开 `role=dialog` 人物详情，底部按钮切换事件，左右箭头支持键盘切换；图片提供 `alt`，关闭按钮提供 `aria-label`。
