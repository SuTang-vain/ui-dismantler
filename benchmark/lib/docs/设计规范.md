# Technical Glossary Explorer - 设计规范

> Domain-neutral benchmark covering 6 view patterns. PC 800×560 / WISE 380×500 / 极端 300×440。

## 2. Tab 结构

- `role="tablist"` + 5 个 `role="tab"` + `aria-selected`/`aria-controls`
- 对应 5 个 `role="tabpanel"`，通过 `hidden` 属性切换
- 活跃 tab 有 `.sg-tab.active` + `border-bottom-color: var(--sg-primary)`

## 3. 交互模式

### Splash
- 全屏 overlay，`.sg-splash.hide` 隐藏
- `.sg-splash-cta` 触发隐藏，ESC 也可关闭

### Quiz
- `.sg-qz-top` 显示题号和题干
- `.sg-opt[data-k]` 选项，点击后显示 `.sg-qz-fb` 反馈
- `.sg-qz-next` 下一题，最后一题后显示 `.sg-qz-result`

### Comparison
- `.sg-cmp-btn` 切换视角
- `.sg-whatif-card.real` + `.sg-whatif-card.alt` 对比卡片

### Graph
- `<svg>` 内 `.sg-node` 节点，点击显示详情
- `NODES` JS 数据驱动

### Nav-Panel
- `<nav class="sg-nav">` + `data-p` 触发器
- `.sg-panel` 子面板切换

### Cause-Chain
- `.sg-timeline-nav` 时间线导航
- `causeChain` JS 数据驱动
- `.sg-whatif-btn` 假设分支

## 4. 逻辑设置

### 响应式断点
- PC: 800×560 (默认)
- WISE: ≤500px -> 380×500，comparison 改纵向
- 极端: ≤320px -> 300×440，tab 字号缩小

### A11y
- tablist/tab/tabpanel 完整 ARIA
- ESC 关闭 splash
- 所有按钮有文字标签

### 零依赖
- 纯原生 JS + CSS，无外部依赖
